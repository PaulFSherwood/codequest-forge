# Books of Knowledge

Books teach concepts. Quests prove concepts.

Each book follows the old three-step teaching rhythm:

1. **Show Me** - a complete working example.
2. **Guide Me** - the same idea with pieces removed.
3. **Test Me** - build from memory without the example.

Books do not award XP directly. They unlock as foundations are completed. Later books can require earlier books to be mastered.

Every book has up to five teaching charges for asking for help. This is an honor-system reminder: use help to learn, not to skip the trial.

Example chain:

```text
Loop Book I
→ Loop Book II: Rows and Columns
→ Array Book II: 2D Maps
```

This prevents jumping from beginner variables into advanced data structures before the foundations are ready.
